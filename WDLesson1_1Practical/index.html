Web Technologies

HTML

HyperText Markup Language. or HMTL, is a standard markup language used to structure content on the web. It uses a system of elements, such as tags and attributes to organize text, images, and links for display in a web browser.HTML acts as the skeleton for a webpage, defining its structure and content.

CSS

Cascading Style Sheets or CSS is a stylesheet language used to add design and formatting to documents written in markup languages like HTML. It controls what a website looks like, including colors, layouts, and fonts, transforming plain text and images into a polished, visually appealing experience. This allows developers to separate the content of a website (managed with HTML) from its presentation, making sites easier to maintain and update.

JavaScript

JavaScript is a dynamic programming language used for creating interactive and dynamic web pages. It is one of the three core web technologies, alongside HTML and CSS r's built-in JavaScript engine executes the code, allowing developers to add features like animations, interactive maps, and updating content without reloading the page.
